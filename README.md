# README.md

## Опис проєкту

**FamilyBudget** — це система для ведення сімейного бюджету. Вона дозволяє контролювати доходи та витрати, аналізувати фінансові звички та планувати бюджет на майбутнє.

### Проблема, яку вирішує система

Багато сімей не ведуть облік витрат, що призводить до хаотичного використання коштів. Додаток спрощує фінансове планування та забезпечує прозорий контроль коштів.

### Основні функції

* Додавання доходів та витрат
* Перегляд статистики та графіків
* Категоризація витрат
* Створення фінансових цілей
* Історія транзакцій

### Для кого продукт

* Для сімей, які хочуть організувати фінанси
* Для студентів, які починають планувати власний бюджет
* Для користувачів, яким потрібен простий спосіб контролювати витрати
